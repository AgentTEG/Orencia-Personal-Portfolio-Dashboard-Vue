# Orencia – Personal Portfolio Dashboard 

🔗 **Live Demo (GitHub Pages):**  
https://agentteg.github.io/Orencia-Personal-Portfolio-Dashboard-Vue/

A **professional dashboard-style portfolio web application** built with **Vue 3 and Vite**.  
This project presents personal, professional, and creative information through a modern and interactive dashboard interface.

---

## 📁 Project Contents

This repository contains a complete Vue.js application with the following main sections:

- **Login Page** – Entry point to the dashboard  
- **Landing Dashboard** – Central navigation page for all sections  
- **Profile Page** – Personal and professional information  
- **Showcase Page** – Highlighted projects and works  
- **Contact Page** – Communication and contact details  
- **Creative Page** – Additional skills and creative tools  

---

## 🚀 Quick Start (Local Setup)

Follow these steps to run the project locally:

1. Clone the repository:
```bash
git clone https://github.com/AgentTEG/Orencia-Personal-Portfolio-Dashboard-Vue.git

2. Navigate to the project directory:

cd Orencia-Personal-Portfolio-Dashboard-Vue


3. Install dependencies:

npm install


4. Start the development server:

npm run dev


The application will be available at:

http://localhost:5173
